# Homelander vs Hitler (2D 横版射击游戏)

这是一个基于 Phaser 3 构建的模块化 2D 横版射击游戏。玩家将扮演“祖国人 (Homelander)”，在充满敌军和障碍的随机关卡中推进，并最终在竞技场挑战邪恶的强大的 Hitler。

## 🚀 核心功能

- **祖国人控制系统**：
  - **自由飞行**：使用 `W/A/S/D` 在全屏范围内无视重力飞行。
  - **持续激光眼**：按住鼠标左键发射高能射线，射线具备实时追踪、掩体阻挡及目标穿透特性。
  - **肉体撞击**：利用飞行惯性撞击敌人可造成巨额物理伤害。
- **动态关卡系统**：
  - **区块化随机生成**：自动拼接平地、高台、战壕、断桥等 5 种不同风格的关卡区块。
  - **可破坏环境**：包含木箱（阻挡）和油桶（连锁爆炸）。
  - **掩体系统**：竞技场内设有可完全抵挡弹幕的坚固掩体。
- **敌军 AI 系统**：
  - **步兵**：平台巡逻并进行水平射击。
  - **坦克**：高 HP 慢速单位，发射重型炮弹。
  - **飞机**：高空横向飞过并投掷炸弹。
  - **炮兵**：固定单位，发射高抛物线弹道。
- **最终 Boss 战 (Hitler Face)**：
  - **表情状态机**：具备 Idle、Hurt、Laughing、Angry 四种动态切换表情。
  - **三阶段战斗**：随着 HP 降低，Boss 会从目光激光进化到咆哮弹幕，最后进入疯狂撞击模式。
- **生存与 UI**：
  - **医疗包**：击败敌人有几率掉落，可恢复生命值。
  - **实时 UI**：左上角血条，右上角得分，屏幕受击反馈。

## 🛠️ 技术栈

- **游戏引擎**：Phaser 3 (Arcade Physics)
- **编程语言**：原生 JavaScript (ES6+)
- **纹理生成**：动态 Graphics API 绘制（无需外部图片资源）

## 📂 文件结构

- `index.html`: 游戏入口，通过 CDN 引入 Phaser。
- `main.js`: 游戏全局配置、场景切换、纹理预加载及摄像机/UI 逻辑。
- `Player.js`: 祖国人玩家类，处理飞行、射线追踪及撞击逻辑。
- `Enemy.js`: 包含敌人基类及四种兵种 AI 逻辑。
- `BossHitlerFace.js`: 最终 Boss 类，处理表情切换及多阶段攻击。
- `LevelManager.js`: 负责随机关卡生成、竞技场触发及道具掉落。
- `Bullet.js`: 处理敌方子弹及追踪导弹逻辑。
- `DestructibleItem.js`: 可破坏环境物件类。

## 🕹️ 如何运行

由于项目使用了 ES 模块化开发，建议通过 Web 服务器运行：

1. **Python 环境 (推荐)**：
   在项目根目录下执行：
   ```bash
   python3 -m http.server 8000
   ```
   然后访问 `http://localhost:8000`

2. **Node.js 环境**：
   可以使用 `http-server`：
   ```bash
   npx http-server
   ```

## 🎮 操作说明

- **移动**：`W/A/S/D` (自由飞行)
- **攻击**：`鼠标左键` (按住发射持续激光)
- **目标**：向右推进至 4500 像素触发 Boss 战，击败 Boss 即可达成任务！
